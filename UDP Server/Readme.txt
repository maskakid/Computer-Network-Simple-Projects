1. Open a command promt and Go to the directory where these two files are stored.
    Example: "cd C:\Python27"
2. give command "python UDPServer.py"
3. Open another command promt and go to that directory
4. Give command "python UDPClient.py"
5. Now give any lowercase input in Client part. Server will return a modified string with uppercase letters. 

*Requires Python 2.7
